# Election_Analysis

## Project Overview
A Colorado Board of Elections employee has given you the following tasks to complete the election audit of a recent local congressional election.

1. Calculate the total number of votes cast.
2. Get a complete list of candidates who received vots.
3. Calculate the total number of votes each candidate received.
4. Calculate the perecentage of votes each candidate won.
5. Determine winner of the election based on popular vote.

## Resources
- Data Source: election_results.csv
- Software: Python 3.7 (64 bit), Visual Studio Code, 1.69.1

## Summary
The analysis of the election show that:
- There were "x" votes cast in the election.

- The canddiate results were:

  - Candidate 3 received "x%" of the vote and "y" number of votes.
- The winner of the election was:
  - Candidate (1, 2, or 3), who received "x%" of the vote and "y" number of votes.
